## 技术栈

```
前端:
  React 18 + TypeScript
  Vite (构建工具)
  Ant Design (UI 组件)
  Zustand (状态管理)

后端:
  Node.js 22 + TypeScript
  Express (Web 框架)
  Socket.io (实时通信)
  Prisma (ORM)

数据库:
  MySQL 8.0 (主数据库)
  Redis (缓存/消息队列)

AI:
  OpenAI SDK (兼容接口)
  支持 5 家模型提供商

部署:
  Docker + Docker Compose
  Nginx (反向代理)
  PM2 (进程管理)
```
