### 工具系统（17 个）

#### 文件操作（7 个）
- `file_read` - 读取文件内容
- `file_write` - 写入文件内容
- `file_list` - 列出目录文件
- `get_file_info` - 获取文件信息
- `parse_excel` - 解析 Excel
- `generate_excel` - 生成 Excel
- `parse_pdf` - 解析 PDF
- `parse_word` - 解析 Word
- `csv_to_json` - CSV 转 JSON

#### 系统命令（3 个）
- `shell_command` - 执行 Shell 命令
- `wait` - 延迟等待
- `http_request` - HTTP 请求

#### 消息通知（3 个）
- `send_message` - 发送消息（微信/钉钉/飞书）
- `broadcast_message` - 批量发送
- `send_task_notification` - 任务状态通知

#### 数据库（5 个）
- `sql_query` - SQL 查询
- `get_table_schema` - 查看表结构
- `export_data` - 数据导出
- `generate_report` - 报表生成
- `db_health_check` - 健康检查

### 测试阶段二：场景测试（Week 2）

#### 2.1 日常办公场景
```
场景：生成周报并发送
1. 用户："分析上周销售数据，生成周报发给团队"
2. Agent 执行：
   - sql_query: 查询销售数据
   - generate_excel: 生成报表
   - send_message: 发送钉钉通知
3. 验证：团队收到带报表的消息
```

#### 2.2 数据分析场景
```
场景：处理 Excel 数据
1. 用户上传 sales_data.xlsx
2. Agent 执行：
   - parse_excel: 解析文件
   - 数据分析（总和、平均、趋势）
   - generate_excel: 生成分析报告
3. 验证：报告内容准确
```

#### 2.3 定时任务场景
```
场景：定时监控报警
1. 配置 Cron: 每天 9:00 执行
2. Agent 执行：
   - db_health_check: 检查数据库
   - 如果异常：send_message 报警
3. 验证：定时触发，异常时报警
```
