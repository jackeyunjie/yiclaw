# OpenClaw 个人 AI 助手 - 需求共识文档

## 1. 需求共识确认

### 1.1 核心需求确认

| 需求项 | 用户确认 | 实现方案 |
|--------|----------|----------|
| AI 模型支持 | ABCD 全支持 + 可视化配置 | 类似 Cherry Studio 的模型管理界面，支持多模型并行、按板块/节点选择 |
| 通道支持 | 微信(个人+企业)、Telegram、飞书 | 优先实现，预留钉钉扩展接口 |
| 安全策略 | Web 登录认证，通道默认开放 | JWT 认证 + 会话管理 |
| 聊天记录保留 | 30 天 | 自动清理策略 |

### 3.2 数据库设计概览

#### 核心表结构
```sql
-- 用户表
users (id, username, password_hash, role, created_at)

-- 会话表
sessions (id, user_id, channel_type, channel_id, title, model_config, created_at)

-- 消息表
messages (id, session_id, role, content, model_used, tokens, created_at)

-- 模型配置表
model_configs (id, name, provider, model_id, api_key, is_default, config_json)

-- 通道配置表
channel_configs (id, channel_type, config_json, is_enabled, webhook_url)

-- 工具配置表
tool_configs (id, tool_name, config_json, is_enabled)
```

### 3.3 模型路由策略

```typescript
// 模型选择优先级：节点配置 > 板块配置 > 全局默认
interface ModelSelectionContext {
  nodeType?: string;      // 节点类型：chat, tool, summary
  section?: string;       // 板块：coding, chat, writing
  messageType?: string;   // 消息类型：text, image, code
}

// 示例配置
const modelRoutingConfig = {
  global: "gpt-3.5-turbo",           // 全局默认
  sections: {
    coding: "claude-3-5-sonnet",     // 代码板块
    writing: "qwen-turbo",           // 写作板块（中文）
    chat: "gpt-3.5-turbo"            // 日常对话
  },
  nodes: {
    tool_call: "gpt-4",              // 工具调用节点
    image_gen: "dall-e-3"            // 图片生成节点
  }
};
```

## 4. 项目边界与限制

### 4.1 明确边界

#### 包含在范围内 ✅
1. Gateway 核心服务（WebSocket + REST API）
2. 多模型支持（OpenAI、Anthropic、国产模型、Ollama）
3. 可视化模型配置中心
4. WebChat 聊天界面
5. 微信通道（个人号 + 企业微信）
6. Telegram Bot 通道
7. 飞书 Bot 通道
8. 基础工具系统（时间、天气、计算器等）
9. Web 管理后台（配置、监控、会话管理）
10. 数据库设计与实现
11. 部署脚本与文档

#### 不包含在范围内 ❌
1. 钉钉接入（Phase 2）
2. 语音交互 ASR/TTS（Phase 2）
3. Canvas 可视化（Phase 2）
4. 浏览器自动化（Phase 2）
5. 移动端 App（Phase 2）
6. 多租户/企业版功能
7. 第三方 OAuth 登录
8. 消息端到端加密

### 4.2 技术约束

| 约束项 | 说明 |
|--------|------|
| 运行环境 | 阿里云 ECS (Windows Server 2022) |
| 数据库 | 阿里云 RDS MySQL 8.0 |
| 开发语言 | TypeScript / Node.js ≥ 22 |
| 部署方式 | PM2 进程管理 |
| 单用户架构 | 专注个人使用，暂不支持多租户 |
